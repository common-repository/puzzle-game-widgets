== Installation ==
This section describes how to install the plugin and get it working.

1. Upload `puzzle-game-widgets` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Use Widgets to place Puzzle game in your Sidebar
4. Or, you can directly place [puzzle-game] shortcode in you post, page or any place you want.
5. Optional parameters for shortcode are: `type` [3x3, 4x4, 5x5] and  `size` [small, medium, large]
